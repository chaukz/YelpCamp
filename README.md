# YelpCamp

YelpCamp is my first full-stack web application that allows users to browse, create, and review campgrounds. Users can register for an account, log in securely, upload images, add campground locations, and leave reviews on campgrounds created by other users. Authentication and authorization ensure that only the owners of campgrounds and reviews can modify or delete their content.

The application demonstrates the use of RESTful routing, CRUD operations, user authentication, server-side validation, session management, and secure web development practices.

## Technologies Used

* Node.js
* Express.js
* MongoDB
* Mongoose
* EJS
* Bootstrap 5
* Passport.js (Local Authentication)
* Express Session
* Connect Flash
* Joi
* Cloudinary
* Multer
* Mapbox
* Helmet
* Express Mongo Sanitize
* Method Override

## Features

* User registration and authentication
* Create, edit, and delete campgrounds
* Upload multiple campground images
* Interactive maps using Mapbox
* Leave and delete reviews
* Authorization for campground and review ownership
* Server-side form validation
* Flash messages for user feedback
* Responsive user interface
